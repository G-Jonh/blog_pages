ES6+ 新特性详解 ​

简介 ​

ES6（ECMAScript 2015）是 JavaScript 语言的一次重大升级，引入了许多新特性。

箭头函数 ​

javascript
const add = (a, b) => a + b

const multiply = (a, b) => {
  return a * b
}

const square = x => x * x

解构赋值 ​

对象解构 ​

javascript
const user = { name: 'YK', age: 25 }
const { name, age } = user
console.log(name) // 'YK'

数组解构 ​

javascript
const numbers = [1, 2, 3]
const [a, b, c] = numbers
console.log(a) // 1

模板字符串 ​

javascript
const name = 'YK'
const greeting = `Hello, ${name}!`

const multiLine = `
  This is a
  multi-line string
`

扩展运算符 ​

javascript
const arr1 = [1, 2, 3]
const arr2 = [...arr1, 4, 5]

const obj1 = { a: 1 }
const obj2 = { ...obj1, b: 2 }

Promise 和 async/await ​

javascript
const fetchData = () => {
  return new Promise((resolve, reject) => {
    setTimeout(() => resolve('data'), 1000)
  })
}

async function getData() {
  const data = await fetchData()
  console.log(data)
}

模块化 ​

javascript
// export.js
export const name = 'YK'
export function greet() { return 'Hello' }

// import.js
import { name, greet } from './export.js'

总结 ​

ES6+ 极大地提升了 JavaScript 的表达能力和开发效率，是现代前端开发的必备知识。
