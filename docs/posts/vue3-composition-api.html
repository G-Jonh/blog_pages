Vue 3 组合式API完全指南 ​

简介 ​

Vue 3 引入了组合式API（Composition API），这是一种全新的代码组织方式，可以更好地组织和复用逻辑代码。

核心概念 ​

setup 函数 ​

setup 是组合式API的入口函数，在组件创建之前执行。

javascript
import { ref, reactive } from 'vue'

export default {
  setup() {
    const count = ref(0)
    const state = reactive({ name: 'YK' })
    
    return { count, state }
  }
}

ref 和 reactive ​

  • ref 用于创建响应式的基本类型数据
  • reactive 用于创建响应式的对象
javascript
import { ref, reactive } from 'vue'

const count = ref(0)
console.log(count.value) // 0

const state = reactive({
  name: 'YK',
  age: 25
})
console.log(state.name) // 'YK'

computed ​

computed 用于创建计算属性。

javascript
import { ref, computed } from 'vue'

const count = ref(0)
const doubleCount = computed(() => count.value * 2)

watch ​

watch 用于监听数据变化。

javascript
import { ref, watch } from 'vue'

const count = ref(0)
watch(count, (newValue, oldValue) => {
  console.log(`count changed: ${oldValue} -> ${newValue}`)
})

最佳实践 ​

组合函数 ​

将相关逻辑封装到组合函数中。

javascript
import { ref, computed } from 'vue'

export function useCounter(initialValue = 0) {
  const count = ref(initialValue)
  
  const increment = () => count.value++
  const decrement = () => count.value--
  const doubleCount = computed(() => count.value * 2)
  
  return { count, increment, decrement, doubleCount }
}

响应式解构 ​

使用 toRefs 将 reactive 对象转换为 ref。

javascript
import { reactive, toRefs } from 'vue'

const state = reactive({
  name: 'YK',
  age: 25
})

const { name, age } = toRefs(state)

总结 ​

组合式API为 Vue 3 带来了更灵活的代码组织方式，特别适合中大型项目。通过组合函数，可以实现逻辑的复用和分离，提高代码的可维护性。
